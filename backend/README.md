# 🎮 Stick Farm Backend

Онлайн сервер для гри Stick Farm з WebSocket підтримкою.

## 📦 Встановлення

```bash
cd backend
npm install
npm start
```

Сервер запуститься на `http://localhost:3000`

## 🚀 Функції

✅ **Реєстрація та автентифікація**
✅ **Спільний баланс на кількох пристроях**
✅ **Реал-тайм синхронізація майнінгу**
✅ **Система друзів**
✅ **Приватні чати**
✅ **Рулетка**
✅ **Трансфери коштів**

## 🔌 WebSocket Events

### Клієнт → Сервер
- `authenticate` - автентифікація
- `start_mining` - запуск майнінгу
- `stop_mining` - зупинка майнінгу
- `transfer_coins` - трансфер коштів
- `send_friend_request` - запит на дружбу
- `accept_friend_request` - прийняття запиту
- `send_private_message` - відправка повідомлення
- `spin_wheel` - крутіння рулетки

### Сервер → Клієнт
- `auth_success` - успішна автентифікація
- `balance_update` - оновлення балансу
- `device_list_update` - список пристроїв
- `new_message` - нове повідомлення
- `spin_result` - результат рулетки

## 💾 Даних

Дані зберігаються у пам'яті серверу. Для прода рекомендується підключити БД (MongoDB/PostgreSQL).

## 🎮 Тестування

1. Запустіть сервер
2. Відкрийте `index-online.html` в браузері
3. Логініться та тестуйте
4. Відкрийте в іншому браузері/пристрої на одному акаунті
5. Баланс синхронізується в реал-тайм! 🚀
